1. örnek

Sinema da koltuk seçimi (paribu cineverse)

Sinema salonunda koltuk seçimi ekranı js eklentisi ile js devre dışı bırakılınca görünmemektedir, çünkü veriler backend den çekilemiyor.
koltuk seçmek veya rezerve edilmiş koltukları tekrar seçmemek için yine js kullanılır.
Örnek kod yazarsak; const container = document.querySelector('.container') (bütün container ı seçtik)
container.addEventListener('click', function(e) { if(e.target.classList.contains('seat') && !e.target.classList.contains('reserved')) { e.target.classList.toogle('selected'); kodlar... } })
yukardaki ufak kod örneğinde container a click olunduğunda koltuk içeren bir div varsa ve bu rezerve değilse onu seçebiliyoruz, selected class ı ile toggle oluşturabiliyoruz.
